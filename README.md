# Projeto PRF - Análise de Acidentes

## Problema
Acidentes em rodovias federais impactam segurança e economia.

## Objetivo
Identificar padrões e fatores de risco com análise de dados.

## Fonte
Base simulada baseada na PRF.

## Dashboard

## Conclusão
Dados ajudam na prevenção e decisão.

## Coleta de Dados
Os dados foram obtidos a partir de uma base simulada inspirada nos dados públicos da Polícia Rodoviária Federal (PRF), estruturados em formato CSV e utilizados para análise no projeto.

## Modelagem
Os dados passaram por tratamento básico, incluindo organização de colunas, padronização de variáveis e utilização de métricas como total de acidentes, mortos e feridos para análise.

## Conclusões
A análise demonstrou que os acidentes estão concentrados em determinadas causas e períodos, com maior gravidade associada ao excesso de velocidade e ocorrências noturnas. Esses resultados podem apoiar ações de prevenção e fiscalização mais direcionadas.
